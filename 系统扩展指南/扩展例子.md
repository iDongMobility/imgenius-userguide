# 扩展例子

## 终端脚本

### 打开作业组直接跳到指定作业

#### 需求描述

作业组中根据指定的作业名跳转到该作业

#### 脚本代码

```js
var curAllTasks = ScriptEngine.Context.GetCurTasks();
if(curAllTasks && curAllTasks.length>0){
	var needJumpTask = _.find(curAllTasks,function(_task){
		return _task.Name == '跳转的作业名';
	});
	if(needJumpTask){
		page.executeTaskTree.JumpTask(needJumpTask);		
	}
}
return true;
