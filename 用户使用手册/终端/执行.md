# 执行

* 点击同步拿到作业组之后，点击作业组名称后面的图标，进入作业组，开始作业，点击下方的下一步图标，根据作业内容进行相应的操作，直到结束，完成巡检。

* 在终端所要执行的内容，一般与日常无终端时所要进行的作业内容一致。根据作业内容的不同，分为以下几种数据类型，需要按照实际情况进行选择或者输入：
  + 列表
  + 数值
  + 字符串
  + 可输入列表
  + 签名
  + 日期时间
  + 文件
  + 多选列表
  + Json字符串

* 同时，在进行选择或输入时，可根据显示样式的不同来进行操作。
