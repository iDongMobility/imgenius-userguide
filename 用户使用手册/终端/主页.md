# 主页

* 登录之后主页内容主要分为作业组的状态操作栏，右上角有信息栏和功能菜单。

* 下方选项相对于登录页面，退出项会转变为日志项，用于日志记录。

* 如果是巡检权限的用户登录，作业组状态操作栏则有待执行、执行中、已执行和创建；如果是有审核权限的用户登录，则会多出一个待审核。
  + 待执行，未开始作业的作业组。
  + 执行中，正在进行作业的作业组。
  + 已执行，已经完成作业，但还未同步至服务器的作业组。
  + 创建，配置了终端可见的即时创建的作业组。
  + 待审核，已经执行同步完，等待审核的作业组。
